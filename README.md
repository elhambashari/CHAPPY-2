
Beskrivning

Chappy-2 är en fullstack-chattapplikation byggd med React, TypeScript och Express.
Användare kan registrera sig, logga in, gå med i offentliga eller privata kanaler och skicka både kanal- och direktmeddelanden.
Gästanvändare kan se offentliga kanaler men kan inte skicka privata meddelanden eller gå med i privata kanaler.


Funktioner

1 Användarregistrering och inloggning med JWT-autentisering

2 Offentliga och privata chattkanaler

3 Direktmeddelanden mellan registrerade användare

4 Gästläge (endast läsbehörighet)

5 Validering av indata med Zod

6 Data lagras i AWS DynamoDB

7 Global tillståndshantering med Zustand

8 Skrivet helt i TypeScript
